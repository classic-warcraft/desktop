## Development
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run electron:serve
```

### Compiles and minifies for production
```
yarn run electron:build
```

### Run your tests
```
yarn run test
```

### Lints and fixes files
```
yarn run lint
```
